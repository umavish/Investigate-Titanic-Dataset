Ananlysis Of Titanic Data¶

Overview¶

This report is an analysis of Titanic data from the dataset on Kaggle website. It contains information about passengers on its maiden voyage including details about their age, class, sex, fare, survival etc. Here I attempt to analyze: 1) How age,sex,class,fare impact survival rate and 2) Which type of passengers were given more preference.

def replace_Embarked(value):
    """
    replacing NaN in Embarked column with most frequent value
    """
    value = np.where(pd.isnull(value),'S',value)
    return value
titanic_df['Embarked']=titanic_df['Embarked'].apply(replace_Embarked)

Adding new column Age_Category

def agesurvival(row):
    if (row['Age']<15):
        val= 'child' 
    elif (row['Age'] >= 15 and row['Age']<=30):
        val='young-adult'
    elif (row['Age'] >= 30 and row['Age']<=50):
        val='middle-aged'    
    else:
        val= 'senior'
    return val

Conclusion¶

Total of 891 passengers data was available. 342 were alive and 549 were dead. 314 females travelled out of which 233 Survived. 577 males travelled out of which 109 survived. It is clear that gender has influence on survival. Chlidren and female are given more preferrence than male. Class also has influence on survival. Almost all females survived from Pclass 1&2. Majority of passengers are from Pclass 3 and their survival rate is also less when compared to others.

Total number of passengers travelled in Titanic was 1,317 but this analysis is made for 891 passengers. And the age of 177 people is missing. If additional information like passengers belong to which deck is provived then this analysis might give much more accurate insights regarding passengers survival.
